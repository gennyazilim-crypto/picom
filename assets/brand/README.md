# Picom App Icon Assets

These are placeholder desktop app icon assets for the MVP Electron foundation.

## Files

- `picom-logo-concept.png`: source logo concept provided by the project owner.
- `app-icon.png`: placeholder app icon copy for Electron/package configuration.
- `app-icon.svg`: original vector placeholder using the approved Picom palette.
- `app-icon.ico`: Windows `.ico` placeholder generated from the Picom app icon.
- `icons/`: generated PNG icon sizes for Linux packaging and future platform icon work.

## Rules

- Do not use Discord logos, icons, copied assets, or exact colors.
- Replace placeholder assets before final production release if needed.
- Keep app icon branding original to Picom.
- Keep UI glyphs separate from brand icons. UI glyphs should use Coolicons through `AppIcon`.
