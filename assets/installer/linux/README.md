# Linux installer assets

Linux AppImage and DEB packages use the original Picom icon set from
`assets/brand/icons`. Desktop-entry branding is generated by electron-builder
from `electron-builder.yml` and uses the Picom product name, the short comment
"Desktop community workspace", and the `Network;Chat;Utility;` categories.

No external or copied brand artwork is used. Add target-specific artwork here
only when AppImage, DEB, or a future RPM target requires a dedicated source.
