# Dark Theme

Picom dark mode is a soft charcoal desktop shell with separated surfaces, muted text hierarchy, and Picom teal/orange accents.

## Token behavior

Dark mode is explicitly defined with `:root[data-theme=dark]`. It keeps the app away from pure black surfaces while preserving clear separation between the server rail, sidebars, chat area, and member list.

## Visual goals

- Soft charcoal background, not pure black.
- Separated elevated surfaces.
- Readable primary/secondary/muted text.
- Subtle borders instead of harsh outlines.
- Teal/aqua active states.
- Orange warning/accent states.
- No Discord colors or copied assets.
