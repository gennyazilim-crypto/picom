TASK 02: Full Profile Page MVP

CONTEXT
User avatars/names currently open a popover or placeholder. Full MVP requires a full profile page inspired by the uploaded profile reference image.

GOAL
Clicking a user avatar/name navigates to a full ProfileView. The profile page must have:
- left profile card
- main profile content
- media gallery
- stats
- bio
- details grid
- skills/tags
- recent activity
- shared media

IMPORTANT
Use the uploaded profile image as structural inspiration only. Do not copy it exactly. Adapt it to Picom community/chat identity.

APP VIEW STATE
activeView:
- "community"
- "mentionFeed"
- "profile"

Add:
- activeProfileUserId
- previousViewBeforeProfile optional

Behavior:
- Clicking user avatar/name sets activeView = "profile" and activeProfileUserId.
- Clicking Back returns to previousViewBeforeProfile if available; otherwise mentionFeed.
- Clicking Home/Picom returns to mentionFeed.
- Clicking community icon returns to community layout and updates activeCommunityId/activeChannelId.

CLICKABLE ENTRY POINTS
- MemberSidebar member avatar
- MemberSidebar member row
- MessageItem avatar
- MessageItem author name
- MentionFeedCard author avatar/name
- Popular People / Following rows in MentionRightPanel or FeedCompanionRail
- UserProfilePopover View profile action if it exists
