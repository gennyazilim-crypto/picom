TASK 01: Feed Companion Rail — Voice Mini Card + Friends + Upcoming Events

CONTEXT
Mention Feed already exists. Mention cards already render. The next requirement is to add a separate right-side companion column aligned with the mention cards.

GOAL
Create a new right-side companion column for Mention Feed cards. It must contain:
1. VoiceMiniControlCard
2. FriendsStatusSection
3. UpcomingEventsSection

CRITICAL LAYOUT RULE
The new column must start at the same vertical position as the first mention card.
It must NOT start next to the Electron titlebar.
It must NOT start next to the story/header/cards area.
It must NOT be placed above the feed.
It must be placed to the right of the mention cards.

REQUIRED STRUCTURE
Inside MentionFeedMain, below story cards and Feed/Takip Ettiğin Kişiler tabs:
