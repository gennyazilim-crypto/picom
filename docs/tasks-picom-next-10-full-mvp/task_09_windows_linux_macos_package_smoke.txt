TASK 09: Windows/Linux/macOS Package Smoke

CONTEXT
Core Full MVP features should be implemented before package smoke. This task verifies desktop packaging readiness.

GOAL
Prepare and run package smoke for:
- Windows
- Linux
- macOS

DO NOT
- add new product features
- redesign UI
- skip honest reporting
- add signing certificates or secrets

CREATE OR UPDATE
- docs/windows-smoke-test.md
- docs/linux-smoke-test.md
- docs/macos-smoke-test.md
- docs/package-smoke-summary.md
- docs/task-checkpoints/task-cross-platform-package-smoke.md

- min size 1100x700
- custom titlebar
- native menu disabled

- nodeIntegration false
- preload path valid
- icon paths valid
- app metadata valid

WINDOWS

- titlebar works

LINUX

MACOS

- icon
- app launches
- titlebar/chrome acceptable
- microphone permission notes
- screen recording permission notes
