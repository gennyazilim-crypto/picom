# Task 100 - Final MVP UI polish against reference image

Picom received a final MVP polish pass focused on the premium desktop reference direction.

## Polish scope

- App frame surface and shadow.
- Header and footer surface feel.
- Message hover background.
- Composer focus treatment.
- Attachment hover feel.
- Overlay shadows.
- Shared focus-visible treatment.
- Reduced-motion fallback for new microinteractions.

## Guardrails preserved

- No mobile UI was added.
- No web-first responsive layout was added.
- No Discord branding, logo, copied assets, or exact colors were added.
- 4-column desktop structure remains unchanged.
- Sidebars remain fixed.
- Chat still scrolls independently.
- Composer remains pinned.

## Manual verification

1. Start the app at 1440x900.
2. Check light mode and dark mode.
3. Switch communities and channels.
4. Hover messages, attachments, channels, and members.
5. Focus composer and buttons with keyboard navigation.
6. Open settings, command palette, context menu, profile popover, and image preview.
7. Confirm no horizontal overflow appears.
