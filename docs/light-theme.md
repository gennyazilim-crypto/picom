# Light Theme

Picom light mode is a soft desktop shell with white main surfaces, subtle borders, dark readable text, and the approved Picom accent palette.

## Token behavior

Light mode is explicitly defined with `:root[data-theme=light]` and keeps parity with the default `:root` values. This makes theme switching deterministic and keeps the UI stable when returning from dark mode.

## Visual goals

- Soft gray outer background.
- White app frame and chat surfaces.
- Subtle sidebar separation.
- Dark readable text.
- Teal/aqua Picom accents.
- No harsh white-on-white layout loss.
- No Discord colors or copied assets.
