# 1. Genel Tasarım Hedefi

Uygulama şu yapıda olacak:

```text
┌────────────┬──────────────────┬──────────────────────────────┬──────────────────┐
│ Server Bar │ Community Sidebar │ Chat / Channel Main Content │ Member Sidebar   │
│ 72px       │ 240px             │ flexible                     │ 260px            │
└────────────┴──────────────────┴──────────────────────────────┴──────────────────┘
```

Ana ekran bileşenleri:

## 1. Sol mini server bar

- Uygulama ana ikonu
- Topluluk/server ikonları
- Yeni topluluk ekleme butonu
- Keşfet butonu
- Ayarlar butonu

## 2. Topluluk sidebar

- Topluluk adı
- Bilgi bölümü
- Kanal kategorileri
- Text kanalları
- Voice kanalları
- Bot/müzik kanalları
- Alt kullanıcı profil kartı

## 3. Orta chat alanı

- Kanal header
- Üst aksiyon ikonları
- Mesaj listesi
- Görsel/media mesaj desteği
- Okunmamış mesaj ayırıcı
- Mesaj yazma inputu

## 4. Sağ kullanıcı paneli

- Arama
- Admin listesi
- Moderatörler
- Katılımcılar
- Online/offline status
- Rol badge’leri

## 5. Tema sistemi

- Light mode
- Dark mode
- Markaya özel accent color
- Tasarım token sistemi

> Not: Discord’u birebir kopyalama. Yapı olarak benzer olabilir ama logo, isim, ikon, spacing ve renkleri kendi markana göre özgünleştir.
